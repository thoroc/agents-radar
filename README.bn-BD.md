## ওয়েব UI

**`PAGES_URL`** — আপনার ফর্কের জন্য ওয়েব UI বেস URL কনফিগার করতে এটিকে একটি রিপোজিটরি ভেরিয়েবল হিসেবে সেট করুন।

একটি পরিচ্ছন্ন, ডার্ক-থিমযুক্ত ইন্টারফেসে সমস্ত পুরোনো ডাইজেস্ট ব্রাউজ করুন — কোনো লগইনের প্রয়োজন নেই। এই রিপোর মার্কডাউন ফাইলগুলো থেকে গিটহাব পেজের মাধ্যমে রিপোর্টগুলো রেন্ডার করা হয়।

ওয়েব UI হলো `packages/web/` ফোল্ডারে থাকা একটি Vite + TypeScript SPA। এটি রানটাইমে `manifest.json` এবং প্রতিটি ডাইজেস্ট ফাইল ফেচ করে — কোনো UI রিবিল্ড ছাড়াই নতুন দৈনিক ডাইজেস্টগুলো প্রদর্শিত হয়।

```bash
bun run dev:web    # start Vite dev server for local UI development
bun run build:web  # build the web UI for deployment
```

## আরএসএস ফিড

**`PAGES_URL`/feed.xml** — নতুন ডাইজেস্ট স্বয়ংক্রিয়ভাবে পেতে যেকোনো আরএসএস রিডারে (Feedly, Reeder, NewsBlur, ইত্যাদি) সাবস্ক্রাইব করুন। ফিড ইউআরএলটি আপনার `PAGES_URL` সেটিং থেকে নেওয়া হয়। ফিডে সব ধরনের রিপোর্টের সর্বশেষ ৩০টি রিপোর্ট অন্তর্ভুক্ত থাকে, যা `manifest.json`-এর সাথে প্রতিদিন আপডেট করা হয়।

## এমসিপি সার্ভার

**`https://agents-radar-mcp.duanyytop.workers.dev`**

একটি হোস্টেড [মডেল কনটেক্সট প্রোটোকল](https://modelcontextprotocol.io) সার্ভার যা এজেন্টস-রাডার ডেটা টুল হিসেবে সরবরাহ করে। যেকোনো এমসিপি-কম্প্যাটিবল ক্লায়েন্ট (Claude Desktop, OpenClaw, ইত্যাদি) সরাসরি সর্বশেষ এআই ইকোসিস্টেম রিপোর্টগুলো কোয়েরি করতে পারে।

**উপলব্ধ টুলসমূহ:**

| টুল | বিবরণ |

|------|-------------|

| `list_reports` | উপলব্ধ তারিখ এবং রিপোর্টের প্রকারের তালিকা (গত N দিন) |

`get_latest` | একটি নির্দিষ্ট প্রকারের সবচেয়ে সাম্প্রতিক রিপোর্টটি আনুন |

`get_report` | তারিখ এবং প্রকার অনুসারে একটি নির্দিষ্ট রিপোর্ট আনুন |

| `search` | সাম্প্রতিক রিপোর্টগুলোর মধ্যে কীওয়ার্ড দিয়ে অনুসন্ধান করুন |

**ক্লড ডেস্কটপ সেটআপ** — `~/Library/Application Support/Claude/claude_desktop_config.json`-এ যোগ করুন:

```json
{
  "mcpServers": {
    "agents-radar": {
      "url": "https://agents-radar-mcp.duanyytop.workers.dev"
    }
  }
}
```
সেভ করার পর ক্লড ডেস্কটপ রিস্টার্ট করুন। এরপর আপনি ক্লডকে এই ধরনের প্রশ্ন করতে পারেন:

- *"এআই সিএলআই টুলসের সর্বশেষ আপডেট কী?"* → `get_latest` কল করে
- *"এই সপ্তাহে ক্লড কোডের উল্লেখগুলো খুঁজুন"* → `search` কল করে
- *"আমাকে ২০২৬-০৩-০৫ তারিখের এআই ট্রেন্ডিং রিপোর্টটি দেখান"* → `get_report` কল করে

**ওপেনক্ল সেটআপ** — নিম্নলিখিত কমান্ডটি চালান:

```bash
openclaw mcp add --transport http agents-radar https://agents-radar-mcp.duanyytop.workers.dev
```
অথবা `~/.openclaw/openclaw.json` ফাইলে এটি ম্যানুয়ালি যোগ করুন:

```json
{
  "mcpServers": {
    "agents-radar": {
      "type": "http",
      "url": "https://agents-radar-mcp.duanyytop.workers.dev"
    }
  }
}
```
এরপর আপনি OpenClaw-কে এই ধরনের প্রশ্ন করতে পারেন:

- *"AI CLI টুলসের সর্বশেষ আপডেট কী?"* → `get_latest` কল করে
- *"এই সপ্তাহে Claude Code-এর উল্লেখগুলো খুঁজুন"* → `search` কল করে
- *"আমাকে ২০২৬-০৩-০৫ তারিখের AI ট্রেন্ডিং রিপোর্টটি দেখান"* → `get_report` কল করে

**সেলফ-হোস্টিং** — `mcp/` ডিরেক্টরি থেকে আপনার নিজস্ব ইনস্ট্যান্স ডেপ্লয় করুন:

```bash
cd mcp
pnpm install
wrangler deploy
```

## বৈশিষ্ট্যসমূহ

- ট্র্যাক করা সমস্ত রিপোজিটরি থেকে গত ২৪ ঘন্টায় আপডেট হওয়া ইস্যু, পুল রিকোয়েস্ট এবং রিলিজ সংগ্রহ করে
- ট্রেন্ডিং ক্লড কোড স্কিলস ট্র্যাক করে — যা সাম্প্রতিকতার পরিবর্তে কমিউনিটি এনগেজমেন্ট অনুসারে সাজানো হয়
- প্রতিটি CLI রিপোজিটরির জন্য টুল-ভিত্তিক সারাংশ এবং বিভিন্ন টুলের মধ্যে তুলনামূলক বিশ্লেষণ তৈরি করে
- একটি গভীর ওপেনক্ল প্রজেক্ট রিপোর্ট এবং ১১টি সমকক্ষ প্রজেক্টের সাথে একটি ক্রস-ইকোসিস্টেম তুলনা তৈরি করে
- সাইটম্যাপের মাধ্যমে অফিসিয়াল অ্যানথ্রোপিক এবং ওপেনএআই ওয়েব কন্টেন্ট স্ক্র্যাপ করে; পর্যায়ক্রমে নতুন আর্টিকেল শনাক্ত করে
- প্রতিদিন গিটহাব ট্রেন্ডিং মনিটর করে এবং ৬টি এআই টপিক ট্যাগ অনুসন্ধান করে; ডাইমেনশন অনুসারে রিপোজিটরিগুলোকে শ্রেণীবদ্ধ করে এবং ট্রেন্ড সিগন্যাল বের করে
- হ্যাকার নিউজ থেকে সেরা ৩০টি এআই স্টোরি সংগ্রহ করে (গত ২৪ ঘন্টা, পয়েন্ট অনুসারে র‍্যাঙ্ক করা); কমিউনিটি সেন্টিমেন্ট রিপোর্ট তৈরি করে
- প্রতিটি রিপোর্টের ধরনের জন্য গিটহাব ইস্যু প্রকাশ করে; `assets/digests/YYYY-MM-DD/` ফোল্ডারে মার্কডাউন ফাইল কমিট করে
- গিটহাব অ্যাকশনসের মাধ্যমে দৈনিক সময়সূচীতে চলে; ম্যানুয়াল ট্রিগারিং সমর্থন করে
- সমস্ত ট্র্যাক করা রিপোজিটরি `config.yml` এর মাধ্যমে কনফিগার করা যায় — কোনো কোড পরিবর্তনের প্রয়োজন নেই
- `locales/*.json` এর মাধ্যমে কেন্দ্রীভূত লোকেল সিস্টেম — `packages/core/src/locales/t.ts` এ `t()` ক্যাটালগ সহ ২১টি ভাষা সমর্থিত

৩. **অ্যাকশনস** ট্যাবে ওয়ার্কফ্লোটি সক্রিয় করুন — এরপর থেকে এটি স্বয়ংক্রিয়ভাবে চলবে

সম্পূর্ণ গাইডের জন্য — সিক্রেটস রেফারেন্স, LLM প্রোভাইডার, `config.yml` কাস্টমাইজেশন, লোকাল ডেভেলপমেন্ট, বহুভাষিক কনফিগারেশন, এবং শিডিউল — [docs/setup.md](./docs/setup.md) দেখুন।

## আউটপুট ফরম্যাট

প্রতিটি সক্রিয় ভাষার জন্য প্রতিটি রিপোর্ট টাইপের একটি করে ফাইল `assets/digests/YYYY-MM-DD/` এ লেখা হবে। ঐতিহাসিক ডাইজেস্টগুলো [`assets/digests/`](./assets/digests/) ফোল্ডারে রয়েছে। সম্পূর্ণ ফাইলের তালিকা, প্রতিটি রিপোর্টের কাঠামো এবং গিটহাব ইস্যু লেবেল রেফারেন্সের জন্য [docs/output-format.md](./docs/output-format.md) দেখুন।
